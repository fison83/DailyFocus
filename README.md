# DailyFocus V5 - 每日专注

一款基于四象限(艾森豪威尔矩阵)的任务管理应用。

## V5 更新内容 (2025-01-07)

### ✨ 新功能

1. **整理按钮优化**
   - 按钮文字简化为"整理"(2字符)
   - 移除整理后的alert弹窗提示

2. **导出文件名时间戳**
   - 文件名格式: `dailyfocus-backup-20250107-143052.json`
   - 避免同一天多次导出文件名冲突

3. **统计页面重新设计**
   - 完成率改为小角标样式,不再占据整个卡片
   - "总任务"、"已完成"、"待整理"三个统计可点击
   - 点击后显示任务详情弹窗,按日期排序

4. **阅读记录表单优化**
   - 采用可折叠卡片设计
   - 所有区域默认展开,点击标题可折叠
   - 避免长滚动,提升填写体验

### 🔧 技术重构

- **模块化架构**: 从单文件HTML重构为模块化结构
- **文件结构**:
  ```
  DailyFocus/
  ├── index.html       # 主页面 (~300行)
  ├── css/
  │   └── style.css   # 样式文件
  ├── js/
  │   ├── config.js   # 配置和更新日志
  │   ├── storage.js  # 数据存储
  │   ├── tasks.js    # 任务管理
  │   ├── goals.js    # 目标管理
  │   ├── reading.js  # 阅读记录
  │   ├── stats.js    # 统计功能
  │   ├── ui.js       # UI交互
  │   └── app.js      # 主入口
  └── README.md
  ```

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 所有数据保存在浏览器localStorage中
3. 使用导出功能定期备份数据

## 主要功能

- 📥 **收集箱**: 快速收集任务想法
- 📊 **四象限**: 基于重要性和紧急性分类任务
- 📈 **统计**: 查看任务完成情况和详情
- 🎯 **目标**: 设定和跟踪长期目标
- 🏷️ **标签**: 自定义标签分类任务
- 📚 **阅读记录**: 记录读书笔记和感想

## 快捷键

- `Enter`: 在快速输入框中添加任务
- `Esc`: 关闭所有编辑面板

## 数据备份

定期使用"导出"功能备份数据,避免数据丢失。
备份文件为JSON格式,可用"导入"功能恢复。
