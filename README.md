# DailyFocus V5

> 基于艾森豪威尔矩阵的四象限任务管理应用

## 快速开始

```
1. 双击打开 index.html
2. 开始使用
```

不需要安装任何东西，没有后端，纯前端应用。

## 主要功能

| 功能 | 说明 |
|-----|------|
| 📥 **收集箱** | 快速记录想法，稍后整理到四象限 |
| 🎯 **四象限** | 按重要/紧急分类管理任务 |
| 📅 **日历** | 按日期查看和管理任务 |
| 📋 **全部任务** | 统一管理所有任务，支持批量操作 |
| ♻️ **回收站** | 已删除的任务，可以恢复或永久删除 |
| 🎯 **目标** | 设定和追踪长期目标进度 |
| ☁️ **云同步** | 使用 GitHub Gist 同步数据（可选） |

## 数据备份

数据保存在浏览器本地存储中。建议定期导出备份：

- **导出**：目标横幅右侧 → 📤 导出
- **导入**：目标横幅右侧 → 📥 导入

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 无依赖：不需要 npm install
- 无构建：直接打开即可使用
